# springjmsactivemq
Spring JMS 结合 ActiveMQ

项目：springjms
springjms 结合 acticemq 学习
Active MQ:apache-activemq-5.14.5
Active MQ管理页面登陆： http://localhost:8161/admin/ ； 
配置用户名和密码  D:\apache-activemq-5.14.5\conf\jetty-realm.properties
